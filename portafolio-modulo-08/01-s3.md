## ¿Qué es S3?
Amazon Simple Storage Service (S3) es un servicio de almacenamiento de objetos escalable y de alta disponibilidad. Permite almacenar y recuperar cualquier cantidad de datos desde cualquier lugar. En este proyecto se utiliza para alojar un **sitio web estático**.

## Pasos realizados (click por click):

### 1. Crear bucket (vía CloudFormation)
El bucket se crea automáticamente con el stack `serverless.yml`.

### 2. Quitar bloqueo público
1. Ir a **S3** en la consola AWS
2. Click en el bucket `serverless-s3bucket-...`
3. Ir a la pestaña **Permissions**
4. En **Block public access** → Click **Edit**
5. Desmarcar todas las opciones → **Save changes**

### 3. Agregar Bucket Policy
En la pestaña **Permissions → Bucket policy**, pegar:
```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadGetObject",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::serverless-s3bucket-lhukubmj6amt/*"
    }
  ]
}
```

### 4. Subir archivo index.html
1. Ir a la pestaña **Objects**
2. Click → **Upload**
3. Subir el archivo `index.html` con el siguiente contenido:
```html
<h1>Proyecto Serverless Inteligente</h1>
```

### 5. Activar Static Website Hosting
1. Ir a la pestaña **Properties**
2. Bajar hasta **Static website hosting**
3. Click → **Edit** → Activar
4. Index document: `index.html`
5. **Save changes**

## Verificar
